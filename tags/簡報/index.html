The easiest way to make slides

no need of Powerpoint or Keynote or any coding

Outline

  1. Workflow
  2. Markdown
  3. Slide generator (Remarkjs and Deckdown)

Workflow

  1. Write your content in Markdown format with horizontal lines, in order to create new pages.
  2. Paste the content to slide generator.

Markdown

  • The philosophy of markdown is to be easy-to-use and produces easy-to-read text.
  • Use header (# in front of the line) or horizontal line (—) to split content into different pages.

Slide generator

Remarkjs

1. Use Remarkise (no need to see the dizzy html tags)

  • Save the content with filename extension “.md”
  • Drag the file to Remarkise

2. Use Remarkjs boilerplate (more geeky way)

  • Paste the content into the body of boilerplate
  • Open the html file with any browser

3. Press “C” to clone the page and “P” to enter presentation mode

* Deckdown

* Just fill the form with content

* Choose a theme and press “Create Deck”

* Press “O” or “Esc” to navigate through slides
