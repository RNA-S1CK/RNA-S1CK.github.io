使用 PlantTFDB 預測胺基酸序列是否為轉錄因子

Dec 21, 2018

Transcriptome 分析的文章中，有時候會出現一個小章節用來描述總共註解到了多少轉錄因子，或是有哪些轉錄因子上調或下調。

因為轉錄因子序列比較短，用一般的註解方式容易錯過，或是僅因為有特定的區段就有把酵素類的蛋白質誤解為轉錄因子的情形。

PlantTFDB 替大家解決了這個問題，網站除了可以直接查詢他們的研究團隊已經預測過的物種中的轉錄因子以外，也有提供了基於他們人為透過文獻閱讀歸納出來的轉錄因子家族指定規則進行預測的服務。

只要輸入胺基酸序列，稍等一下就可以得到預測結果。以下將會簡單介紹網頁以及紀錄預測功能的使用，並且銜接到使用 excel 繪製轉錄因子比例圓餅圖的操作。

首先首頁就可以看到五十八種轉錄因子的分類以及已經註解到的數量，每次改版過程中註解方式或是說指定規則可能會因爲文獻而略有更動，像是最近一版就把參與轉錄調節但是不具有直接 DNA 結合區段的類別移除了。

隨便選擇一個沒有聽過的 E2F/DP 點進去，有該種轉錄因子在各種物種中的家族成員數量，和簡單的文字敘述及引用文獻，讓人可以快速了解該分類，或是本身在進行該家族的基因研究的話，也可以參考不同物種中的成員數量以評估註解情形。

接下來終於要進入正題，如何使用ＴＦＤＢ提供的轉錄因子註解服務：

  1. 進入預測的頁面：

  2. 輸入 fasta 格式的序列或是直接上傳 fasta 檔案

  3. 勾選 Best Hit in Arabidopsis Thaliana 的話會附上一筆資訊，未來銜接其他分析也方便，然後用力地按下 Pridiction

  4. 過一下下就有結果出來了

  5. 在文字敘述中按 Download list

  6. 下載下來的是純文字檔案，可以用 Excel 打開

  7. 在預測結果上方插入一個 row 輸入 ＴＦ，旁邊插入一個 column 輸入 count

  8. 將 count 填滿 1

  9. 選取這兩個我們處理過的 column，點選 insert 分頁中的 Pivot Table

  10. 在喜歡的地方放這個新 Table

  11. 在右方把 count 和 TF 都勾選起來

  12. 依據數量大小排個序

  13. 就可以畫圓餅圖啦
